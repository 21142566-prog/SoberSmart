# Sober Smart Web

This project is now a plain static web app that can be opened directly with Live Server without Vite or a build step.

## Run locally

1. Open `index.html` in VS Code.
2. Start Live Server.
3. Open the served page in your browser.

## Notes

- No `npm install` or `npm run dev` is required for testing.
- The page includes a basic Web Serial button for connecting to STM32 over USB.
- If your browser does not support the Web Serial API, the app will show a warning.
